# wcalculator

Calculadora de anilhas para barra. Informe o peso total desejado e a barra — o app calcula quais anilhas colocar em cada lado.

## Desenvolvimento

```bash
cd frontend
npm install
npm run dev
```

## Build

```bash
cd frontend
npm run build
# output em frontend/out/
```

## Testes

```bash
cd frontend
npm test          # unitários (Vitest)
npm run test:e2e  # integração (Playwright)
```
